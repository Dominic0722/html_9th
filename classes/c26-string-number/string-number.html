문자 데이터 - string

문자데이터를 표현하기 위해서는 내용의 앞뒤로 쌍따옴표("")/홀따옴표('')/백틱(``)를 붙여서 표현한다.

문자데이터끼리 더하면 이어진다.

숫자데이터 - number

숫자데이터는 문자데이터와 같은 쌍따옴표 표식 없이 숫자만 입력하면 된다. 단, 유효한 숫자여야 한다. (예)0100과 같은 숫자는 숫자데이터로 인식되지 않으므로 100이라고 입력해야한다.
숫자데이터끼리 사칙연산이 가능하다.
문자와 숫자를 더하면 이어진다.(더하기만 됨) 그결과는 문자데이터로 정한다

``백틱으로 문자데이터를 표현하면 문자데이터 안에서 줄바꿈을 허용한다.

"문자데이터".length 라고 쓰면 앞의 문자데이터의 문자개수를 계산하여 그 자리에 남긴다.

"문자데이터"[순번]라고 쓰면 문자데이터안에서 순번에 맞는 문자를 불러와 그 자리에 남긴다.(0부터 숫자를 입력할 수 있다.)
