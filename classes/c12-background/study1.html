배경 - background

배경은 컨텐츠가 없거나 높이가 없으면 배경표현이 되지 않는다.

배경스타일이 들어가는 공간은 태그의 가장 밑바닥이다.(컨텐트박스보다 아래깔린다)

background-color: brown; padding: 20px; height: 300px; width: 300px; box-sizing: border-box;

background-color

배경색상 입력
태그의 공간(테두리 안쪽 모든 공간)에 배경색을 나타낼 때 사용하는 css속성

height: 80vh; background-color: bisque; background-image: url("../img/elijah-grimm-8y5O0ZSbbqw-unsplash.jpg"); background-repeat: no-repeat; background-position: 100px 200px; margin: 100px 0;

background-image

배경이미지 입력 - url("")도구 사용
배경이미지는 입력하면 자동으로 가로세로로 반복된다.

background-repeat

배경이미지가 반복되는 속성

repeat(기본값 - 가로세로 반복)

no-repeat

repeat-x

repeat-y

background-position

배경이미지의 위치를 변경할 때 사용하는 속성

background-position: x좌표 y좌표

키워드방식,%방식,픽셀방식

height: 100vh; background-color: deeppink; background-image: url("../img/elijah-grimm-8y5O0ZSbbqw-unsplash.jpg"); background-repeat: no-repeat; background-size: 300px auto; /* 가로300 세로는 비율에 맞게 auto */ background-size: auto 100%; background-size: cover; /* 태그의 공간을 가득채우는 방식 - 배경이미지가 잘려서 안보일수 도 있다 */ background-size: contain; /* 이미지가 태그의 공간안에 잘리지않는 선에서 최대크기로 들어가서 채우는 방식 */ background-position: center center; background: aquamarine url("../img/elijah-grimm-8y5O0ZSbbqw-unsplash.jpg") no-repeat 0 0 /cover; /* color url repeat position size */

cover

cover방식은 배경이미지크기를 이미지가 잘려나가도 태그의 공간을 가득 메우는 방식

contain

contain방식은 배경이미지가 잘려나가지 않는 선에서 배경이미지의 크기를 최대한 키워서 넣는 방식

축약형 표현

background:color url repeat position / size(순서중요!)
